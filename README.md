# USA-Real-Estate-Sales-Dashboard

## Overview

Designed an interactive Power BI dashboard to analyze the U.S. housing market using over 1 million property listings.

## Tools Used

- Power BI
- Power Query
- DAX

- ## Dashboard Features

- Interactive state-level filtering
- KPI Cards
- Property status analysis
- Bedroom distribution analysis
- Average property price by category
- Top 10 states by average property price
- Dynamic filtering using slicers

## Skills Demonstrated

- Data Cleaning
- Data Transformation
- DAX
- Dashboard Design
- Data Visualization
- Business Intelligence
